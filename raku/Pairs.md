# Pairs

* There is a class called Pair
* The pair class does the associative role
* There are two parts to a pair
    * key
    * value
* Syntaxes for creating pairs:
    * Pair.new('key', 'value');
    * 'key' => 'value';
    * :key<value>
    * :key<value1 value2>;
        * this creates a pair with a list containing to values for its value
* Chained pairs
