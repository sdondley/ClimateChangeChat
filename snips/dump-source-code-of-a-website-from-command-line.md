# Dump Source Code of a Website from Command Line

* for macOS only
* will return the page with javascript processing done

## Using Google Chrome
```
 /Applications/Google\ Chrome.app/Contents/MacOS/Google\ Chrome –headless –dump-dom https://example.com
 ```
